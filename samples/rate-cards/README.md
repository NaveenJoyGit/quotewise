# Sample rate cards for onboarding testing

Upload these in **Step 2 — Rate Card** at `/onboarding`.

| File | Work type to select | Format |
|------|---------------------|--------|
| `painting.txt` | Painting | Plain text (recommended) |
| `painting.csv` | Painting | CSV |
| `false-ceiling.txt` | False Ceiling | Plain text |

## Quick test

1. Complete Step 1 (business profile).
2. Choose **Painting** and upload `painting.txt`.
3. Click **Parse with AI**.
4. Review the rate table, edit if needed, then **Save Pricing & Continue**.

Use `false-ceiling.txt` with work type **False Ceiling** for the second work type (via `manage-rates` after onboarding, or repeat onboarding flow).
